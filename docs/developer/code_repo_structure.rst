Project Code Repository Structure
=================================

The **Renku** project code structure is split into the following github
repositories.

- renku_ - build and deploy the platform
- renku-authorization_ - Resource Manager service
- renku-commons_ - scala service commons
- renku-deployer_ - Deployer service
- renku-explorer_ - Knowledge Graph explorer
- renku-graph_ - Knowledge Graph and type systems
- renku-identity_ - Identity Management and foreign policies (TBD)
- renku-python_ - Python API & Command Line Interface (CLI)
- renku-storage_ - Storage service
- renku-tutorials_ - files for getting started application (TBD)
- renku-ui_ - web front-end interface
